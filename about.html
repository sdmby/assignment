📌 项目简介

本项目是一个静态的数据可视化Web应用，旨在展示中国主要城市的GDP数据和经济发展趋势。通过直观的图表和交互功能，帮助用户更好地理解城市经济发展情况。

项目采用纯HTML、CSS和JavaScript构建，无需后端服务器，可直接部署在GitHub Pages等静态托管服务上。

🎯 主要功能

  • ✅ 城市GDP排名展示
  • ✅ 多年GDP增长趋势图
  • ✅ 城市详细信息查看
  • ✅ 多城市对比分析
  • ✅ 搜索和筛选功能
  • ✅ 响应式设计，支持移动端

📊 数据说明

数据项 说明 单位
GDP 地区生产总值（国内生产总值） 亿元人民币
人口 常住人口数量 万人
面积 行政区域面积 平方公里
增长率 GDP同比增长百分比 %

⚠️ 注意：本项目数据仅供学习参考，实际数据请以官方统计公报为准。

📖 使用说明

部署到GitHub Pages

  1. 将项目代码推送到GitHub仓库
  2. 进入仓库 Settings → Pages
  3. 选择 Source 为 main 分支
  4. 等待部署完成即可访问

本地运行

由于使用了Fetch API加载JSON数据，需要通过本地服务器运行：

# 使用 Python
python -m http.server 8000

# 或使用 Node.js (需安装 http-server)
npx http-server
